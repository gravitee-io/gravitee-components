# Logos

This UI library contains Gravitee.io logos in different flavors and colors:
 - 2 sizes: full width (the default one) and a square one
 - 3 colors: cyan, dark blue and white

It also includes a favicon and a Gif loader.
All theses images are available in the published NPM package inside the `@gravitee/ui-components/assets/images` folder.

To use them in your app you can either copy paste the images or reference them directly in your CSS file like this:
```css
.spinner {
    ...
    background-image: url('../node_modules/@gravitee/ui-components/assets/images/gravitee-loader.gif');
    background-repeat: no-repeat;
    background-position: center center;
}
```

⚠️ If you're using Webpack (or another builder) do not forget to add some configuration to be sure the images from `@gravitee/ui-components/assets/images` will well end in the final bundle
