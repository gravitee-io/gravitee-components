# Console Colors

All the colors defined in this palette are available as CSS properties.
To use them you just need to import `gravitee-console-theme-v2-color`:

```css
@import 'xxxx/@gravitee/ui-components/assets/css/gravitee-console-theme-v2-color.css';
```
